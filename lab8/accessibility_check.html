Lab 1: Accessibility

Website Section 508 and WCAG 2.0 Accessibility Check

Major118.com


This website contains information and profiles for the rap group major 118 and it failed the accessibility check the failures that came up were "1.1 Text Alternatives: Provide text alternatives for any non-text content" which can be fixed by adding an alt attribute to the area element. The alt attribute should describe the link destination and make sense when read out of context, "2.1 Keyboard Accessible: Make all functionality available from a keyboard" which can be fixed by adding an onblur handler to the script that performs the same function as the onmouseout handler, and "3.1 Readable: Make text content readable and understandable." which can be fixed by adding the HTML documents add the lang attribute.

Blacktie.es


This website features the blacktie design companies portfolios and services and it failed the accessibility check and the failures that showed up were "1.1 Text Alternatives: Provide text alternatives for any non-text content" which can be fixed by Add an alt attribute to there img element, and "1.4 Distinguishable: Make it easier for users to see and hear content including separating foreground from background" that can be fixed by incorporating a better color ratio between the foreground and background.

W3C.org


The World Wide Web Consortium website is an international community where Member organizations, a full-time staff, and the public work together to develop Web standards and it passed the accessibility check.
